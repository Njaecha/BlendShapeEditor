This plugin is a Addon to Shallty's [BlendShapeCreator](https://github.com/ShalltyB/BlendshapeCreator) which provides an ingame editor.

It is based on GhostEndSky's [KKShapeEditor](https://github.com/Njaecha/GhostEndSky_KKShapeEditor) with most parts 
outside the editor stripped away for better compatibility with the KK plugin ecosystem.
